API DOCUMENTATION

The Neureon API provides real-time synthetic identity detection for credit applications. This documentation covers authentication, endpoints, and integration examples.

Base URL https://api.neureon.group

Overview

Neureon uses a hybrid detection engine combining rule-based pattern matching with machine learning to identify synthetic identities. The API analyzes SSN structure, credit behavior, device signals, and application velocity to generate risk scores and recommendations.

Key capabilities:

  • Real-time Detection — Sub-100ms response times for inline decisioning
  • Batch Processing — Scan up to 500 records per request for portfolio review
  • SAR Generation — Auto-generate FinCEN-compliant narratives
  • Webhooks — Real-time alerts to Slack, Discord, or any HTTP endpoint

Authentication

All API requests require an API key passed in the X-API-Key header. Contact us to obtain your production API key.

HTTP Header 
X-API-Key: your_api_key_here
⚠️
Keep your API key secure. Do not expose it in client-side code or public repositories.

Quickstart

Here's a complete example detecting fraud on a credit application:

cURL 
curl -X POST https://api.neureon.group/api/v2/detect \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your_api_key" \
  -d '{
    "ssn": "123-45-6789",
    "credit_score": 720,
    "credit_file_months": 8,
    "authorized_user_accounts": 6,
    "phone_carrier": "Mint Mobile",
    "is_vpn": false
  }'
Python 
import requests

response = requests.post(
    "https://api.neureon.group/api/v2/detect",
    headers={"X-API-Key": "your_api_key"},
    json={
        "ssn": "123-45-6789",
        "credit_score": 720,
        "credit_file_months": 8,
        "authorized_user_accounts": 6
    }
)

result = response.json()
print(f"Risk: {result['risk_assessment']['combined_score']}")
print(f"Decision: {result['risk_assessment']['recommendation']}")
JavaScript 
const response = await fetch('https://api.neureon.group/api/v2/detect', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    ssn: '123-45-6789',
    credit_score: 720,
    credit_file_months: 8,
    authorized_user_accounts: 6
  })
});

const result = await response.json();
console.log(`Risk: ${result.risk_assessment.combined_score}`);

Fraud Detection

The primary detection endpoint analyzes an application and returns a risk assessment.

POST /api/v2/detect

Request Parameters

Parameter Type Description
ssn required string Social Security Number (any format accepted)
credit_score integer Applicant's credit score (300-850)
credit_file_months integer Age of credit file in months
authorized_user_accounts integer Number of AU tradelines
phone_carrier string Mobile carrier name
address_history_count integer Number of previous addresses
applications_24h integer Applications in last 24 hours
is_vpn boolean VPN detected on connection
is_emulator boolean Device emulator detected

Response

JSON Response 
{
  "status": "success",
  "application_id": "a1b2c3d4e5f6",
  "risk_assessment": {
    "combined_score": 0.847,
    "category": "high",
    "auto_deny": true,
    "recommendation": "AUTO-DENY"
  },
  "patterns": {
    "total_triggered": 3,
    "pattern_ids": [
      "V1-IMPOSSIBLE_CREDIT_VELOCITY",
      "V1-AU_TRADELINE_STUFFING",
      "V1-HIGH_RISK_CARRIER"
    ]
  },
  "response_time_ms": 47
}

Decision Values

DecisionRisk ScoreRecommended Action
APPROVE< 0.20Standard processing
REVIEW0.20 - 0.35Manual analyst review
STEP-UP0.35 - 0.60Additional verification required
DENY0.60 - 0.85Decline application
AUTO-DENY> 0.85Automatic decline, add to watchlist

Batch Processing

Scan multiple records in a single request. Ideal for portfolio reviews.

POST /api/v2/batch
Request 
{
  "records": [
    {"id": "member_001", "ssn": "123-45-6789", "credit_score": 720},
    {"id": "member_002", "ssn": "987-65-4321", "credit_score": 680}
  ],
  "include_patterns": true
}
ℹ️
Maximum 500 records per batch request. SSNs are masked in responses for security.

Rate Limits

The API is rate limited to 60 requests per minute per IP address. Batch endpoints count as a single request regardless of record count.
