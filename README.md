# Neureon Group Landing Page

Next-generation fraud prevention platform that outperforms traditional banking security.

## Quick Deploy to Vercel

### Option 1: Deploy via GitHub (Recommended)

1. Create a new GitHub repository
2. Upload these files:
   - `index.html` (the landing page)
   - `package.json`
   - `vercel.json`
   - `README.md`

3. Go to Vercel and click "Continue with GitHub"
4. Select your new repository
5. Click "Deploy"

### Option 2: Deploy via Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. In this project directory, run:
   ```bash
   vercel
   ```

3. Follow the prompts to deploy

### Option 3: Drag & Drop

1. Go to [vercel.com/new](https://vercel.com/new)
2. Simply drag this entire project folder into the browser
3. It will automatically deploy

## Custom Domain

Once deployed, you can add your custom domain `neureon.group` in Vercel's project settings.

## Files Included

- `index.html` - The main landing page
- `package.json` - Project configuration
- `vercel.json` - Vercel deployment settings
